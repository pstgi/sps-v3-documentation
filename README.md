The SPS Plus project is in continuous development. New releases include new features, bug fixes,
and other improvements. Information about the different versions of SPS Plus app can be found in the
[Change Log](./CHANGELOG.md) file. 

Currently, the updates are performed manually. Request an update from the PSTGI team to get the
